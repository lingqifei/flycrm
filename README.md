#FLY-CRM
07FLY-CRM是零起飞网络开发的，基于GPLv3协议发行，针对中小企业管理活动，提供客户关系管理员(CRM)，进销存(JXC),人力资源(HRM)，后勤(办公用品，固定资产，公物维修)、物业管理等功能，软件开源免费，争取打造中国开源CRM软件第一品牌。
