###  系统说明： 


07FlyCRM客户管理系统，基于GPLv3协议发行，针对中小企业管理活动，目前提供crm客户管理，erp进销存，oa办公，hrm人事，fms财务等软件开源，争取打造中国开源CRM软件第一品牌.
该系统适用各类行业，主要针对中小型销售/售后服务团队。
该系统支持客户资料自定义配置、查询以及支持二次开发接口。
如果你想拥有一个简单实用、接地气的客户管理系统，也许是您最佳的选择。
